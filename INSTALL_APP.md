# Installer FasoStock comme application

FasoStock est une **PWA** (Progressive Web App). Vous pouvez l’installer sur **mobile** et **desktop** pour un accès direct, comme une application native.

## Sur téléphone (Android)

1. Ouvrez FasoStock dans **Chrome** (ou Edge).
2. Un bandeau « Installer FasoStock » peut s’afficher en haut → appuyez sur **Installer**.
3. Sinon : menu du navigateur (⋮) → **« Installer l’application »** ou **« Ajouter à l’écran d’accueil »**.
4. L’icône FasoStock apparaît sur l’écran d’accueil ; l’app s’ouvre en plein écran sans barre d’adresse.

## Sur iPhone / iPad (iOS)

1. Ouvrez FasoStock dans **Safari**.
2. Appuyez sur le bouton **Partager** (carré avec flèche).
3. Choisissez **« Sur l’écran d’accueil »**.
4. Nommez l’app si besoin, puis **« Ajouter »**.
5. L’icône apparaît sur l’écran d’accueil ; en l’ouvrant, l’app s’affiche en mode standalone.

## Sur ordinateur (Windows / macOS / Linux)

1. Ouvrez FasoStock dans **Chrome** ou **Edge**.
2. Dans la barre d’adresse, cliquez sur l’icône **« Installer »** (⊕ ou « Installer l’application »).
3. Ou : menu (⋮) → **« Installer FasoStock »**.
4. L’app s’ouvre dans sa propre fenêtre (sans onglets du navigateur).

## Après installation

- L’app se met à jour automatiquement quand vous déployez une nouvelle version.
- Les données restent celles de votre compte (connexion Supabase) ; l’installation ne change pas vos données.
